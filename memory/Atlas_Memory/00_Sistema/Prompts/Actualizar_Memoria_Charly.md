# Protocolo de Actualización de Memoria

Al finalizar una conversación importante, analizar si surgió información que merezca conservar.

## Clasificación
Clasificar la información en una de estas categorías:
- **Perfil** (datos personales, estilo de aprendizaje)
- **Memoria_Activa** (decisiones recientes, estado actual)
- **Aprendizajes** (errores cometidos, lecciones aprendidas)
- **Decisiones** (opciones elegidas y por qué)
- **Ideas** (proyectos, reflexiones)
- **Diario** (reflexiones personales)
- **Proyectos** (en curso)
- **Universidad** (materias, fechas de examen, notas)
- **Salud** (si es relevante)
- **Finanzas** (si es relevante)

## Proceso
Para cada elemento encontrado:

1. **Resumir** la información en menos de 150 palabras
2. **Indicar** el archivo donde debería guardarse
3. **Explicar** por qué vale la pena conservarla
4. **Preguntar confirmación** a Charly: "Voy a guardar esto en [categoría], ¿dale?"
5. **NO escribir automáticamente** el archivo sin confirmación

## Reglas
- Nunca guardar conversaciones completas
- Solo conservar conocimiento útil a largo plazo
- Priorizar información relevante para el examen del jueves
- Si Charly dice "no", no guardar
- Si Charly dice "sí", guardar y confirmar: "✅ Guardado en [archivo]"
