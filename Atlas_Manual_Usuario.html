1. Introducción

Atlas es un asistente personal inteligente que corre localmente en tu computadora. A diferencia de los chatbots tradicionales, Atlas tiene características únicas:

  • 🧠 Memoria a largo plazo: Aprende sobre vos con el tiempo
  • 🤖 Múltiples agentes: Se adapta según lo que necesités
  • 👁️👂🗣️ Sentidos: Ve tu pantalla, escucha tu voz, te habla
  • 🌐 Busca en internet: Cuando necesita información actualizada
  • 🔒 100% privado: Tus datos nunca salen de tu PC
  • 🪞 Auto-conocimiento: Puede analizar su propia arquitectura
💡 Lo más importante: Todo corre en tu computadora. No hay nube, no hay servidores externos. Tu información personal está 100% protegida.

3. Instalación

Opción A: Instalador Automático (Recomendado)

  1. Descargá el archivo Atlas_v2.0_YYYYMMDD.zip
  2. Descomprimilo en una carpeta (ej: C:\Atlas\)
  3. Click derecho en Atlas_Installer.bat → "Ejecutar como administrador"
  4. Esperá 10-20 minutos
  5. Se crearán accesos directos en tu escritorio
✅ El instalador hace automáticamente:
  • Instala Python 3.11 si no lo tenés
  • Instala Ollama (el cerebro de Atlas)
  • Descarga el modelo qwen3:8b (~4.7 GB)
  • Instala Tesseract OCR (para leer imágenes)
  • Instala todas las dependencias de Python
  • Crea accesos directos en el escritorio

Opción B: Instalación Manual

# 1. Instalar Python 3.11+ desde https://python.org
# 2. Instalar Ollama desde https://ollama.com
# 3. Descargar el modelo
ollama pull qwen3:8b
# 4. Instalar Tesseract OCR
# 5. Instalar dependencias de Python
cd Atlas
pip install -r requirements.txt
# 6. Ejecutar Atlas
python run.py

8. Agentes Inteligentes

Atlas tiene 5 agentes especializados. No necesitás llamarlos manualmente, Atlas decide cuál usar según tu pregunta.

🗣️ Agente General

Cuándo se activa: Conversación casual, preguntas generales

Características:

  • Estilo casual pero respetuoso
  • Usa conocimiento general del modelo
  • No carga material de estudio

Ejemplos: "hola, ¿cómo estás?", "¿Quién es Elon Musk?"

📊 Agente Estadística

Cuándo se activa: Preguntas sobre estadística, probabilidad, exámenes

Características:

  • Carga tu material de estudio
  • Traduce fórmulas a lenguaje natural
  • Empieza con ejemplos concretos

Ejemplos: "¿Qué es la covarianza?", "Resolvé este ejercicio"

🌐 Agente Researcher

Cuándo se activa: Información actualizada, noticias, eventos específicos

Características:

  • Busca en internet (DuckDuckGo)
  • Cita fuentes con URLs
  • Información actualizada

Ejemplos: "¿Quién es el presidente actual?", "Últimas noticias sobre IA"

💭 Agente Psicólogo

Cuándo se activa: Emociones, consejos personales, estrés

Características:

  • Empático y cálido
  • Escucha activa
  • No juzga

Ejemplos: "me siento estresado", "necesito hablar"

🏛️ Agente Arquitecto Mental

Cuándo se activa: Razonamiento profundo, metacognición

Características:

  • Muestra solo una "nube" de 1-2 líneas
  • Respuesta directa e insightful
  • Sin mostrar el proceso lógico

Ejemplos: "ayudame a pensar sobre mi carrera", "analizá este problema"

9. Sentidos de Atlas

👁️ Ojos: Captura de Pantalla

Uso: !mirar

Qué hace:

  1. Captura tu pantalla actual
  2. Extrae texto con OCR (Tesseract)
  3. Muestra el texto extraído
  4. Opcionalmente lo analiza con Atlas

Casos de uso: Leer PDFs que no se pueden copiar, analizar apuntes en imagen.

👂 Oídos: Reconocimiento de Voz

Uso: !escuchar o !escuchar [segundos]

Qué hace:

  • Con internet: Usa Google Speech (muy preciso)
  • Sin internet: Usa Vosk (offline, ~50MB)

🗣️ Voz: Síntesis de Voz

Uso: !hablar on/off

Qué hace:

  • Con internet: Edge TTS (Microsoft, alta calidad)
  • Sin internet: pyttsx3 (offline, calidad básica)

Voces disponibles:

  • Español Argentina: es-AR-ElenaNeural
  • Español México: es-MX-DaliaNeural
  • Español España: es-ES-ElviraNeural

Listá todas con: !voces

11. Auto-conocimiento y Mejora

!autoconocer

Atlas analiza su propia arquitectura y muestra:

  • Estructura de archivos
  • Capacidades activas (ojos, oídos, voz)
  • Estado de salud del sistema
  • Dependencias instaladas
  • Métricas (archivos indexados, historial)

!reflexionar

Atlas analiza sus conversaciones pasadas y genera una reflexión honesta:

  • Dónde fue más útil
  • Dónde falló
  • Patrones detectados
  • Qué debería mejorar

!mejorar

Atlas busca en la web mejores prácticas para:

  • RAG (Retrieval Augmented Generation)
  • Prompt engineering
  • Optimización de modelos locales

12. Sistema de Perfiles

¿Qué es un perfil?

Un perfil es un paquete con:

  • Tus prompts personalizados
  • Tu memoria (conversaciones, aprendizajes)
  • Tu configuración

Exportar tu perfil

!exportar_perfil charly

Genera un archivo ZIP con toda tu configuración. Podés:

  • Hacer backup
  • Migrar a otra PC
  • Compartir con amigos

Importar un perfil

!importar_perfil ruta/al/archivo.zip

Restaura un perfil previamente exportado. Siempre hace backup antes de importar.

Crear perfil nuevo

!nuevo_perfil juan

Crea una estructura de memoria vacía para otro usuario.

14. Solución de Problemas

Atlas no arranca

Problema: ModuleNotFoundError
Solución: pip install -r requirements.txt

Ollama no responde

Problema: Atlas dice que Ollama no está disponible
Solución:
  1. Verificá que Ollama esté corriendo: ollama list
  2. Si no está, iniciá el servicio Ollama
  3. Verificá el modelo: ollama pull qwen3:8b

El micrófono no funciona

Problema: !escuchar da error
Solución:
  1. Probá el micrófono: !probar_mic
  2. Verificá permisos en Windows: Configuración → Privacidad → Micrófono
  3. Instalá pyaudio correctamente

La voz no suena

Problema: !hablar on no reproduce audio
Solución:
  1. Verificá que pygame esté instalado: pip install pygame
  2. Probá con voz offline: pip install pyttsx3

Atlas no encuentra información

Problema: busca [tema] no encuentra nada
Solución:
  1. Reconstruí el índice: !indexar
  2. Verificá que los archivos estén en memory/Atlas_Memory/

La UI no abre

Problema: streamlit run atlas_ui.py da error
Solución: 
pip install streamlit
streamlit run atlas_ui.py --server.port 8501
Luego abrí manualmente: http://localhost:8501
